## Введение

Проект представляет собой простую реализацию сервиса для загрузки и просмотра фотографий.
Пользователи могут регистрировать аккаунты, загружать и удалять свои фотографии и отмечать понравившиеся фотографии других пользователей.

## Ограничения

При анализе решения были выявлены следующие особенности, подлежащие доработке в следующих версиях:
1. Предоставленный фронтенд работает с удаленными API:
* https://nomoreparties.co/
* https://auth.nomoreparties.co/
2. https://nomoreparties.co/ не реализует полноценную аутентификацию, т.к. работает с подставляемой из кода заглушкой токена. Наличие jwt-токена, полученного из auth.nomoreparties.co проверяется только на фронте.
3. предоставленный фронтед не может без доработки приложения вцелом быть подключен к предоставленному бэкенду, развернутому локально, т.к. 
* бэкенд требует авторизации с помощью jwt
* ожидает куки, а фронтенд хранит токен в LocalStorage
* предоставляет пути к API, не включающие в себя GroupId
* не обрабатывает cors
4. Указанные в пункте 3 проблемы не будут исправлены в ходе работы над текущей версией, предполагается запускать только фронтенд-приложение в связке с сервисом nomoreparties.
